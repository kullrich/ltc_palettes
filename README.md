## Install 
Install the package using the following commands  <img align="right" src="logo.png" width=300>

`library(devtools)`

`install_github("loukesio/ltc_palettes")`

## Palettes

Select the palette of interest: e.g., "paloma" 
wiht the function `paloma <- ltc("paloma")`
and print it with 
`pltc(paloma)`
